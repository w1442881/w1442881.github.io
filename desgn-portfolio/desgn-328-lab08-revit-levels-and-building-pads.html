Notes

Instructions

1. Open a new project file in Revit using the Default template

Property lines

  1. Find the Massing and Site tab
    1. Select Property Lines from the Modify Site Panel
    2. Select Create by entering distances and bearings
  2. Enter the property lines as found on the site plane
    1. I suggest starting at the bottom left corner and working clockwise
    2. Insert line to close instead of manually inputting the last property line
      1. Make sure the automatically created line is similar to what is on the site plan

Topography

Create a new topography using the elevations of the corner points shown on the site plan as we have done in previous labs Use the pick lines tool to make your boundary from the property lines You will have to manually create this surface by placing points I suggest subtracting 100' from the elevations

Levels

Use an elevation view to add 2 levels Open an Elevation View Find the Architecture Tab Select Level from the Datum Panel Draw two new levels Change the elevation to match what is given in the survey drawing Rename one Level "Driveway" Rename the other level "Foundation" Change the view scale so the level markers are easily visible Change the end points and markers so they are wider than the surface, and in line with each other Delete Level 2 Rename Level 1 to Topography Pin levels to avoid accidentally editing them

Import CAD

Open the level 1 floor plan view Change Visual Style to Wireframe Select Insert CAD from the Import Panel of the Insert Tab Insert the Property Lines File Unpin file to allow it to be moved Edit the scale property to adjust the size until it matches your property lines Properties menu Edit Type A scale factor of 12 is pretty close Add detail lines Open the Annotate tab and find Detail Lines in the Detail Panel Use the Pick Lines tool to add detail lines around the building pad and roadway

Building pads

Open the Office Foundation floor plan view Open the Massing and Site Tab Select the Toposolid drop down arrow Select Create from Sketch Make the Foundation set the level to Office Foundation Use the sketch tools to make the outline of your building pad Change the type to 8" Concrete Make the Driveway As above, but use the Driveway Level Change Pad Thickness Click on the Foundation Select Edit Type in the properties palette Select Duplicate Change Name to Foundation - 8" Concrete Click on the Driveway Select Edit Type in the properties palette Select Duplicate Change Name to Driveway - 10" Concrete Select edit structure Change concrete thickness to 6"

Edit Topography

Select Topography Select Modify Sub Elements Place points to create flat topography under the foundation and driveway that is 6" lower than the concrete Elevations Open a floor plan view Find the View Tab Select Elevation Practice Adding Elevation Views Make a New North, South, East and West Rename your views Have one elevation cut through a building pad Section Views Open a floor plan view Find the View Tab Select Section Make a section view through a build pad Make another section view and practice using split segments Make another section practice changing the view range try making it shorter try making it narrower Look at your section view after each change to see what changes

Sheets

Find Sheets in the project Browser Right Click Sheets and Select New Sheet Use your ARC template Make 2 new Sheets Name one Sections and the other Elevations Drag and drop your section views into the proper sheet Drag and drop your elevation views onto their proper sheet Practice scaling and moving these views Save a PDF of both sheets
