Autodesk Revit 2024

Procedures

Open a new file in Revit:

  1. Open Revit ✔
  2. Select New..., under models on the left side of the screen ✔

Open a new file in Revit using a Default project template:

  1. Follow instructions above to open a new file ✔
  2. Select default.rte ✔

Find the Massing & Site tab:

Select theProperty lines button in Massing & Site tab.

Select theCreate by entering distances and bearings button in Property lines buton in the Massing & Site tab.
